# Memory Changelog

메모리 파일의 모든 변경 이력. `memory-curator`가 자동 작성한 메모리 업데이트 PR이 머지될 때마다, 그리고 사람이 수동 편집할 때마다 한 줄씩 추가된다.

## 포맷

```
YYYY-MM-DD | PR#<번호> | <파일> | <요약>
```

---

```
seed       | PR#seed  | conventions.md | 시드 항목 CONV-001~008 생성

```
